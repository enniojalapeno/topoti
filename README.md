# Topoti

The all-in-one productivity app combining todo list, pomodoro timer, and timeline.

## Features

- **Todo List** - Manage your tasks
- **Pomodoro Timer** - Focus timer with work/break intervals
- **Timeline** - Visual timeline of your activities

## Usage

Open `todo.html` in your browser to start using the app.

## Tech Stack

- HTML, CSS, JavaScript
